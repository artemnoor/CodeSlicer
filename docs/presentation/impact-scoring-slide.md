# Граф не просто находит связи — он оценивает их

_Слайд размещается после демонстрации impact analysis и перед результатами
тестирования._

```text
Impact Score                 Chain Confidence                 Token Saving
Что проверять                Насколько достоверна             Сколько лишнего кода
в первую очередь             цепочка                          не передаётся ИИ
```

```text
ImpactScore(v) = Criticality(v) * Confidence(path) * Decay^distance
```

Impact Score — приоритет узла по критичности, достоверности пути и расстоянию.
Chain Confidence — геометрическое среднее confidence всех рёбер цепочки.
Token Saving — измеренная доля контекста, исключённая графом из передачи ИИ.

Не показывать на этом слайде неподтверждённые проценты. Если измерения
контекста отсутствуют, использовать подпись: «Потенциальное сокращение
передаваемого контекста».
